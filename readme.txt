Typing Race Game
==========================

Overview:
---------
Simple Type Racer game using Python and the Pygame library.
Four lines are displayed at a time, and when time is up, 
your Words Per Minute (WPM) score is calculated and displayed.

Features:
---------
- Text passages from famous works.
- A 10-second timer that starts on the first keystroke.
- Visual feedback: Correct letters turn green and incorrect letters turn red.
- A "Play again?" button appears after the game ends.

Installation:
-------------
1. Ensure Python 3 is installed on your system.
2. Install the required packages using pip. In the project directory, run:
   
   pip install -r requirements.txt

Running the Game:
-----------------
1. Open a terminal or command prompt.
2. Navigate to the directory containing the game files.
3. Run the game with:
   
   python game.py

Controls:
---------
- Type normally to input text.
- Use the Backspace key to correct mistakes.
- When time is up, your WPM score is displayed and a "Play again?" button appears. Click the button to restart.

Enjoy the game and happy typing!
